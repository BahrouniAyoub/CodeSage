# CodeSage

A web application that leverages artificial intelligence to explain code snippets in an easy-to-understand manner.

## Overview

CodeSage is an interactive tool designed to help developers understand complex code by providing AI-powered explanations. Whether you're learning to code, reviewing legacy code, or trying to understand a new library, this tool breaks down code logic into simple, digestible explanations.

## Features

- 🤖 **AI-Powered Explanations** - Get instant, intelligent explanations for any code snippet
- 💻 **Code Syntax Highlighting** - Beautiful syntax highlighting for improved readability
- 🎨 **Clean & Intuitive UI** - User-friendly interface designed for seamless experience
- ⚡ **Real-time Processing** - Get explanations instantly as you paste your code
- 🔄 **Multiple Language Support** - Works with various programming languages
- 📱 **Responsive Design** - Works perfectly on desktop and mobile devices

## Technology Stack

- **Frontend**: JavaScript (80.4%)
- **Styling**: CSS (15%)
- **Markup**: HTML (4.6%)

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection

### Installation

1. Clone the repository:
```bash
git clone https://github.com/BahrouniAyoub/AI-Code-Explainer.git
```

2. Navigate to the project directory:
```bash
cd AI-Code-Explainer
```

3. Open `index.html` in your web browser:
```bash
open index.html
# or for Windows
start index.html
```

Alternatively, serve the project using a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.

## Usage

1. **Paste or type code** into the code input area
2. **Select the programming language** (if multiple are supported)
3. **Click "Explain"** to get AI-powered explanation
4. **Review the explanation** in the results panel
5. **Adjust settings** as needed for different explanation styles
